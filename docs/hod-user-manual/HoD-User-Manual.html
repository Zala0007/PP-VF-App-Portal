HoD Quick User Guide

Visiting Faculty Application Portal — visitingfaculty.ldcecollege.org

1. Open HoD Login

Open the website → scroll to the footer → click Admin Login → enter the HoD password → click Sign In.
Click Admin Login (red box)
Enter the password and click Sign In

2. Mark the Application Reviewed

Click the orange Pending button. It changes to green Reviewed.
Click Pending in the red box
Reviewed, View Details, and Select status

3. View Candidate Details

Under Actions, click View Details and check the complete application.
Check education, experience, subjects, availability, and documents

4. Select One Status

Choose one status
StatusMeaning
RejectedNot selected
Shortlisted for InterviewEnter interview details
SelectedCandidate selected
Interview details screen

5. Select Official Google Account

Select your official departmental email account, not a personal account.
Select the official HoD account

6. Review and Send

Edit the pre-filled email if needed, verify the details, and click Send.
Verify From, To, subject, and message
The candidate receives the email after you click Gmail's Send button.

Repeat these steps for all applications.
